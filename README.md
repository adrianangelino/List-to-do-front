# Lista de Tarefas

Este é um projeto de Lista de Tarefas desenvolvido utilizando React e Vite. O objetivo é permitir que os usuários gerenciem suas tarefas de forma eficiente, adicionando, marcando como concluídas e removendo-as conforme necessário.

## Funcionalidades

- Adicionar novas tarefas
- Marcar tarefas como concluídas
- Remover tarefas
- Interface amigável e responsiva

## Tecnologias Utilizadas

- React
- Vite
- JavaScript
- CSS

## Instalação e Execução

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/adrianangelino/List-to-do-front.git
   ```

2. **Navegue até o diretório do projeto:**

   ```bash
   cd List-to-do-front
   ```

3. **Instale as dependências:**

   ```bash
   npm install
   ```

4. **Execute o servidor de desenvolvimento:**

   ```bash
   npm run dev
   ```

   O aplicativo estará disponível em `http://localhost:3000`.

## Estrutura do Projeto

O projeto segue a seguinte estrutura de diretórios:

```
List-to-do-front/
├── public/
│   └── index.html
├── src/
│   ├── components/
│   │   └── TodoItem.jsx
│   ├── App.jsx
│   └── main.jsx
├── package.json
└── vite.config.js
```

- **`public/`**: Contém o arquivo HTML principal.
- **`src/`**: Contém os arquivos fonte do React.
  - **`components/`**: Contém os componentes React, como `TodoItem.jsx`.
  - **`App.jsx`**: Componente principal do aplicativo.
  - **`main.jsx`**: Ponto de entrada do React.
- **`package.json`**: Contém as dependências e scripts do projeto.
- **`vite.config.js`**: Configurações do Vite.
